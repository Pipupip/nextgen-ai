# NextGen AI — корпоративный сайт AI-агентства

Корпоративный сайт для агентства по автоматизации бизнеса и внедрению нейросетей.

## Страницы

- **Главная** — hero, услуги, процесс работы, статистика, отзывы, FAQ
- **Услуги** — детальное описание с ценами и калькулятором стоимости
- **Портфолио** — кейсы с фильтрацией
- **Блог** — статьи про автоматизацию и AI

## Особенности

- Тёмная корпоративная тема (сине-фиолетовый градиент)
- Калькулятор стоимости услуг
- Карусель отзывов
- FAQ-аккордеон
- Интеграция с Telegram (приём заявок)
- SEO-оптимизация
- Адаптивный дизайн

## Стек

- HTML5 / CSS3 / JavaScript (vanilla)
- Netlify (хостинг)

## Запуск

Просто откройте `index.html` в браузере.
